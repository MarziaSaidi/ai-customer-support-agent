# Return Policy

**Last updated:** July 2026

## Overview

We want you to love your purchase. If something isn't right, you can return eligible items within **60 days** of delivery.

## Eligible items

- Shoes and apparel in **unused condition** with original tags attached
- Items must be in resalable condition (no wear, scuffs, or damage)
- Final sale and customized products are **not** eligible for return

## How to start a return

1. Log in to your account and open **Orders**
2. Select the order and choose **Start a return**
3. Print the prepaid shipping label (if provided)
4. Drop off the package at any authorized carrier location

## Refund timing

- Refunds are processed within **5–7 business days** after we receive your return
- The refund goes back to your original payment method
- You'll receive an email when the refund is issued

## Exchanges

Need a different size? Start a return and place a new order for the size you need. Exchanges ship separately once your return is in transit.
